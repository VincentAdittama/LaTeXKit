## Outline Structure

- [ ] Introduction section planned (hook → context → thesis → roadmap)
- [ ] Body sections have clear themes/arguments

- [ ] Logical flow between sections established
- [ ] Argument balance checked (similar depth across major sections)

## Citation Integration
- [ ] Bibliography file (e.g., `<your-export>.bib`) loaded and validated

- [ ] Citations mapped to claims requiring support
- [ ] Multiple citations used for well-established claims
- [ ] `[SOURCE NEEDED]` flags added where evidence is missing
- [ ] All citation keys match exactly with .bib file

## Quality Standards
- [ ] Scope adheres to assignment requirements
- [ ] Word count estimation per section provided
- [ ] Estimated total length within ±20% of target
- [ ] Evidence density appropriate for academic level
- [ ] Transitions between sections explicit
- [ ] All major claims have supporting evidence planned

## Writing Roadmap
- [ ] Sections that can be written independently identified
- [ ] Sections requiring additional research flagged
- [ ] Connections between sections noted
- [ ] Suggested writing order provided

## Constitution Compliance
- [ ] Outline supports original thinking
- [ ] Proper attribution planned throughout
- [ ] Balanced use of sources (no over-reliance)
- [ ] Argument coherence validated

## Deliverables
- [ ] Outline file versioned correctly (YYYYMMDD_v01_outline.md)
- [ ] Estimated word counts per section included
- [ ] Citation statistics provided (total citations, unique sources)
- [ ] Notes added for complex arguments
